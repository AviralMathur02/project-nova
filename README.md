# Project Nova

Project Nova is a voice-first AI study companion designed to help learners prepare for technical interviews through natural conversations, adaptive learning, and personalized feedback.

## Vision

Build an AI assistant that doesn't just answer questions—it teaches, quizzes, remembers progress, and helps users improve over time.

## Tech Stack

- React
- Spring Boot
- PostgreSQL
- OpenAI APIs
- Whisper
- Docker
